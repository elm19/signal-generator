The Trading Time Series Forecast API is a Flask-based REST API providing access to machine learning models for trading predictions and analysis. It's designed for ease of use, allowing you to quickly retrieve predictions and model performance metrics.

## API Base URL

The API is deployed and hosted on PythonAnywhere. You can access it via the following base URL:

```
http://your-pythonanywhere-username.pythonanywhere.com
```

#### Example Request

```python
import requests

# Make GET request to the root endpoint
response = requests.get('http://your-pythonanywhere-username.pythonanywhere.com/')
print(response.json())
```

#### Example Response

```json
{
    "api_status": "healthy",
    "available_endpoints": [
        {
            "path": "/",
            "description": "API information and status"
        },
        {
            "path": "/predict",
            "description": "Get trading predictions"
        },
        {
            "path": "/model-info",
            "description": "Get information about available models"
        }
    ],
    "version": "1.0.0"
}
```

The response includes:
- `api_status`: Current health status of the API
- `available_endpoints`: List of all available API endpoints with descriptions
- `version`: Current API version number
