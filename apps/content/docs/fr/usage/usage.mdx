## À Travers l'Interface Utilisateur Future (Planifiée)
Dans une phase de développement future, une interface utilisateur dédiée sera mise en œuvre. Cette interface fournira un moyen plus intuitif et visuel de consommer les signaux de trading et de surveiller les performances des modèles.

L'interface prévue inclura :

- Un tableau de bord affichant les signaux actuels et les niveaux de confiance.
- Des graphiques de performance historique des signaux.
- Des informations sur les métriques des modèles individuels et les résultats des backtests.
- Potentiellement, des options de configuration pour les utilisateurs.

Cette méthode sera idéale pour :

- **Traders** : Qui préfèrent une vue d'ensemble visuelle et une interaction directe sans codage.
- **Utilisateurs non techniques** : Qui ont besoin de signaux exploitables en un coup d'œil.

Des détails sur cette intégration future peuvent être trouvés dans la [Feuille de Route d'Intégration Future](/docs/development/roadmap).

## Développement Local et Personnalisation
Pour ceux qui s'intéressent aux aspects techniques, à l'extension des fonctionnalités ou simplement à l'exécution du projet dans un environnement local, le code source du générateur est disponible pour une configuration locale et une personnalisation.

Cette méthode est idéale pour :

- **Développeurs** : Modifier les modèles, ajouter de nouvelles fonctionnalités ou expérimenter avec différentes sources de données.
- **Apprenants** : Ceux qui veulent approfondir et comprendre comment le modèle a été créé.

Pour des instructions sur la configuration locale du projet, consultez [Développement et Contribution](/docs/development/local-setup).
