# Getting Started

Welcome to the Gold Futures Signal Trading Generator documentation. This guide will help you set up and start using the trading signal generation platform.

## Prerequisites

Before you begin, ensure you have the following installed:
- Python 3.8 or higher
- pip (Python package manager)
- Git

## Installation

1. Clone the repository:
```bash
git clone https://github.com/elm19/GoldSpot-Predictor
cd GoldSpot-Predictor
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Quick Start

1. Configure your API credentials:
```python
# config.py 
API_KEY = "your_api_key"
API_SECRET = "your_api_secret"
```

2. Initialize the signal generator:
```python
from goldspot.generator import SignalGenerator

generator = SignalGenerator()
signals = generator.generate_signals()
```

3. View the trading signals:
```python
print(signals.latest())
```

## Configuration

The signal generator can be configured through a configuration file or environment variables:

- `TIMEFRAME`: Trading timeframe (e.g., "1h", "4h", "1d")
- `INDICATORS`: List of technical indicators to use
- `RISK_LEVEL`: Risk management parameters (1-5)

See the [Configuration](/docs/configuration) section for detailed settings.
