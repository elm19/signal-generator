description: Explication de haut niveau du Générateur de Signaux de Trading sur l'Or
---

Le Générateur de Signaux de Trading sur l'Or est un système robuste conçu pour générer des signaux de trading prédictifs (par exemple, "achat", "vente", "maintien") pour les contrats à terme sur l'or. Son objectif principal est d'automatiser et d'améliorer les décisions de trading grâce à une approche systématique basée sur les données. Le projet est construit autour d'une API REST basée sur Flask, qui sert d'interface principale pour accéder à ces prédictions. L'API s'appuie sur un pipeline de données sophistiqué qui collecte, prétraite et crée des caractéristiques à partir des données historiques des contrats à terme sur l'or, qui alimentent ensuite des réseaux de neurones récurrents (RNN) comme les LSTM et les GRU. Chaque modèle subit des tests de backtesting rigoureux pour garantir sa viabilité et fournir des métriques de performance complètes.

## Public Cible

Cette documentation est adaptée à différents utilisateurs : les développeurs intéressés par l'intégration de l'API, les traders à la recherche d'insights sur les signaux, et toute personne curieuse des aspects techniques du générateur.
