# API Documentation

The Gold Futures Signal Trading Generator provides a RESTful API that allows you to access trading signals and historical data programmatically.

## Authentication

All API requests require authentication using an API key and secret. Include these in the request headers:

```bash
Authorization: Bearer your_api_key
X-API-Secret: your_api_secret
```

## Base URL

```
https://api.goldspot-predictor.com/v1
```

## Endpoints

### Get Latest Signal

```http
GET /signals/latest
```

Returns the most recent trading signal.

**Response**

```json
{
  "timestamp": "2024-01-20T14:30:00Z",
  "symbol": "XAUUSD",
  "signal": "STRONG_BUY",
  "price": 2023.45,
  "confidence": 0.89,
  "indicators": {
    "rsi": 67.8,
    "macd": 1.23,
    "moving_average": 2020.15
  }
}
```

### Get Historical Signals

```http
GET /signals/history
```

Query Parameters:
- `start_date`: ISO timestamp
- `end_date`: ISO timestamp
- `limit`: Number of records (default 100)

**Response**

```json
{
  "signals": [
    {
      "timestamp": "2024-01-20T14:30:00Z",
      "symbol": "XAUUSD",
      "signal": "STRONG_BUY",
      "price": 2023.45,
      "confidence": 0.89
    }
  ],
  "metadata": {
    "total_count": 1,
    "page": 1
  }
}
```

## Error Handling

The API uses standard HTTP status codes:

| Code | Description |
|------|-------------|
| 200  | Success |
| 400  | Bad Request |
| 401  | Unauthorized |
| 403  | Forbidden |
| 429  | Too Many Requests |
| 500  | Internal Server Error |

Error Response Format:

```json
{
  "error": {
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "API rate limit exceeded",
    "details": {
      "retry_after": 60
    }
  }
}
```

## Rate Limiting

- Free tier: 60 requests per minute
- Pro tier: 1000 requests per minute

Rate limit headers are included in all responses:
```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1516131640
```
